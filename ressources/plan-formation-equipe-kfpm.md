# Plan de formation equipe KFPM

## Objectif

Former les gerants, personnels, stagiaires et collaborateurs a l'usage utile de l'IA dans l'atelier PUB-KFPM x LN-IA.

La formation doit produire des livrables reels, pas seulement des exercices abstraits.

## Publics concernes

- gerants ;
- personnel administratif ;
- equipe commerciale ;
- stagiaires ;
- collaborateurs LN-IA ;
- professeurs impliques dans la communication ou l'orientation.

## Competences visees

1. Comprendre le role de l'IA dans le projet.
2. Rediger un brief clair.
3. Utiliser un prompt de production.
4. Corriger un contenu IA.
5. Respecter la validation institutionnelle.
6. Publier avec methode.
7. Suivre les resultats.
8. Produire un mini-rapport.

## Module court - 2 heures

### Partie 1 - Vision et regles

Durée : 20 minutes

Contenu :

- pourquoi KFPM utilise l'IA ;
- difference entre vitesse et qualite ;
- role de KPM, FPM et KFPM ;
- chaine de validation.

Livrable : reformulation par chaque participant du principe directeur.

### Partie 2 - Ecrire un bon brief

Durée : 20 minutes

Contenu :

- objectif ;
- public cible ;
- message central ;
- format ;
- canal ;
- responsable ;
- date ;
- indicateur.

Livrable : un brief complet pour une publication reelle.

### Partie 3 - Produire avec IA

Durée : 30 minutes

Contenu :

- utiliser la banque de prompts ;
- demander plusieurs variantes ;
- eviter les informations inventees ;
- adapter le ton au public.

Livrable : une premiere version de message, affiche ou publication.

### Partie 4 - Corriger avec methode

Durée : 25 minutes

Contenu :

- verifier l'exactitude ;
- simplifier le texte ;
- distinguer KPM, FPM et KFPM ;
- supprimer les promesses excessives ;
- preparer la validation.

Livrable : une version corrigee.

### Partie 5 - Publier et suivre

Durée : 15 minutes

Contenu :

- choisir le canal ;
- noter la publication ;
- suivre reactions et leads ;
- inscrire les decisions.

Livrable : une ligne completee dans un tableau de suivi modele.

### Partie 6 - Mini-bilan

Durée : 10 minutes

Contenu :

- ce qui est pret ;
- ce qui reste a valider ;
- prochain responsable ;
- prochaine date.

Livrable : mini-rapport de fin d'atelier.

## Exercices de production reelle

Chaque participant choisit un livrable :

- message WhatsApp parents ;
- publication KPM ;
- publication FPM ;
- relance commerciale ;
- fiche formation IA ;
- tableau de suivi ;
- mini-rapport hebdomadaire ;
- brief d'affiche.

## Evaluation

La formation est validee si chaque participant produit un document :

- utile ;
- corrige ;
- classe dans le bon dossier ;
- pret a validation ;
- relie a un objectif mesurable.

## Suite recommandee

Organiser un atelier hebdomadaire court de 45 minutes :

- 10 minutes : choix des productions ;
- 20 minutes : production et correction ;
- 10 minutes : validation ou points bloquants ;
- 5 minutes : suivi des indicateurs.
