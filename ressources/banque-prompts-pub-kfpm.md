# Banque de prompts PUB-KFPM

## Objectif

Fournir des prompts prets a utiliser pour produire des affiches, messages, relances, rapports, tableaux et supports de formation dans le cadre de PROJET-PUB-KFPM x LN-IA.

## Regles d'utilisation

- Toujours preciser KPM, FPM ou KFPM.
- Toujours donner le public cible.
- Ne jamais demander a l'IA d'inventer une information institutionnelle.
- Toujours relire avant publication.
- Toujours classer la production dans le bon dossier.

## Prompt 1 - Clarifier un brief

```text
Tu es assistant de production pour PROJET-PUB-KFPM x LN-IA.

Aide-moi a clarifier ce brief avant production.

Entite concernee : [KPM / FPM / KFPM]
Objectif : [objectif]
Public cible : [public]
Message central : [message]
Format souhaite : [affiche / WhatsApp / publication / rapport / tableau]
Canal : [canal]
Date prevue : [date]
Responsable : [nom]
Indicateur de resultat : [indicateur]

Retour attendu :
1. brief corrige ;
2. points manquants ;
3. risques de confusion ;
4. recommandation de production.
```

## Prompt 2 - Message WhatsApp parents

```text

Entite : [KPM / FPM]
Sujet : [sujet]
Public : parents d'eleves ou candidats
Objectif : [informer / inviter / relancer / confirmer]
Information verifiee a inclure : [information]

Contraintes :
- ton respectueux et rassurant ;
- pas de promesse excessive ;
- message court ;
- distinction claire entre KPM et FPM si necessaire.

Propose 3 versions.
```
