# 00_GENERER_DANS_ATELIER

Ce dossier contient les fichiers a generer, partager ou remplir pendant l'atelier Formation IA KFPM.

## Fichier disponible

- `formulaire-individuel-membre-formation-ia-kfpm.xlsx`

## Usage

1. Envoyer une copie du fichier Excel vierge a chaque membre.
2. Chaque membre remplit son fichier individuellement.
3. Le fichier rempli est retourne au responsable formation.
4. Le responsable ajoute uniquement les emails autorises dans `assets/js/config.js`.
5. Les fichiers remplis restent dans un espace prive.

## Regle importante

Les fichiers remplis contiennent des informations personnelles. Ils ne doivent pas etre publies dans le depot public ni sur GitHub Pages.
